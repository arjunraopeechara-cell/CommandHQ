# CommandHQ - Organization Management Platform

A minimalist Mac OS-style organization management platform built with clean, simple UI/UX principles.

## Features

### üéØ Core Features
- **Mac OS-style Dock Navigation** - Bottom navigation bar with smooth interactions
- **Customizable Dashboard Widgets** - Drag and organize your workspace
- **Progressive Disclosure** - Click to expand details, keeps interface clean
- **Event Management** - Track events through 5-stage pipeline
- **Task Management** - Create and assign tasks with status tracking
- **Budget Control** - Manage budgets with line items
- **Multi-Level Approvals** - Configure approval workflows
- **Member Management** - Role-based team organization
- **Leadership Handoff** - Structured transition templates
